🤖 Echo Bot – ChatGPT Telegram Bot

📌 Overview

Echo Bot is a Telegram chatbot powered by OpenAI’s GPT model that enables real-time conversational AI inside Telegram. It maintains context-aware responses and allows users to interact naturally using simple commands.

Built using:
⚡ aiogram (Telegram Bot Framework)

🧠 OpenAI GPT API

🔐 Environment-based configuration

🚀 Features

💬 AI-Powered Conversations

Responds intelligently using OpenAI’s GPT model

🧠 Context Awareness

Maintains conversation history for better responses

🧹 Clear Chat Memory

Reset conversation context with a command

📖 Help Command
Built-in guide for users

⚡ Fast Async Handling

Uses aiogram for efficient asynchronous message processing

🛠️ Tech Stack:
Python 🐍
OpenAI API
aiogram
python-dotenv

⚙️ Setup & Installation
1️⃣ Clone the repository
git clone https://github.com/your-username/echo-bot.git
cd echo-bot

2️⃣ Install dependencies
pip install aiogram python-dotenv openai

3️⃣ Create .env file
OPENAI_API_KEY=your_openai_api_key
TELEGRAM_BOT_TOKEN=your_telegram_bot_token

▶️ Running the Bot
python main.py
